# 初中物理提分宝典 - 公式手册与专项突破

这是一款专为中国初中生设计的**高保真、强交互式物理学习网页**，整合了力学、电学、热学、声学和光学等板块的核心物理公式、定义、典型例题及专项习题，并支持极高分辨率的 A4 打印级 **PDF 导出**。

## 🌟 核心特色

1. **高奢现代视觉排版**：支持深色/浅色模式自适应，融合现代化毛玻璃（Glassmorphism）、微交互悬浮卡片及细腻的展开过渡动画。
2. **KaTeX 毫秒级公式渲染**：全局数学与物理公式均采用极速 KaTeX CDN 进行矢量渲染，无论屏幕放大多少倍均清晰无比。
3. **智能求解与单位换算器**：在任意公式卡片的“换算计算”选项卡中，输入任意已知量并选择单位（如将时间设为分钟、小时，或速度设为 km/h），点击“智能求解”即可自动进行单位换算，输出详尽的 LaTeX 代入求解步骤！
4. **专项习题及双模式 PDF 导出**：
   - 精选力、电、热学经典压轴计算题，网页端支持一键展开/折叠名师步骤解析。
   - **学生打印版**：导出 PDF 时全自动隐藏所有答案解析，并在题目下方生成精致的**灰色虚线答题空白区**，填空题生成横线，完美匹配 A4 打印纸张，便于线下闭卷手写做题。
   - **教师解析版**：导出 PDF 时展示最完整的公式替换与推导步骤，方便批改或核对答案。

## 🛠️ 本地预览与测试

本项目采用极简设计，零编译依赖，在本地即可秒级开箱即用。

### 方式一：直接双击运行
直接在浏览器中打开项目根目录下的 `index.html` 文件即可启动！

### 方式二：使用本地轻量服务预览 (推荐)
如果您安装了 Node.js，可以在项目根目录下启动一个本地静态服务器预览：
```bash
# 方式 A：使用 npx 快速启动
npx serve .

# 方式 B：使用 Python 快速启动
python -m http.server 8000
```
启动后在浏览器中打开 `http://localhost:3000` (或 8000) 即可访问。

## 🚀 部署在 Cloudflare (CF) 上

本项目非常适合部署在 **Cloudflare Pages** 静态托管服务上。我们已经为您预先配置好了 `wrangler.toml`，您可以选择以下任一方式进行部署：

### 方式一：使用 Wrangler CLI 命令行（一键秒级部署）

如果您拥有 Cloudflare 账号，只需在终端中运行以下命令，即可直接将当前目录的网页发布到 CF 节点：

```bash
# 首次运行会提示登录您的 Cloudflare 账号
npx wrangler pages deploy .
```

*部署完成后，命令行会直接输出一个形如 `https://xxx.pages.dev` 的专属公网访问链接！*

### 方式二：关联 GitHub 仓库自动部署（持续集成，推荐）

1. 在您的 GitHub 上新建一个代码仓库，并将本项目的所有文件推送上去。
2. 登录 [Cloudflare Dashboard](https://dash.cloudflare.com/)。
3. 点击左侧导航栏的 **“Workers & Pages”** $\rightarrow$ **“Create application”** $\rightarrow$ **“Pages”**。
4. 选择 **“Connect to Git”**，关联您的 GitHub 账号并选择刚刚创建的代码仓库。
5. 在构建设置中：
   - **Framework preset (框架预设)**: 选择 `None`。
   - **Build command (构建命令)**: 保持为空（不需要构建）。
   - **Build output directory (输出目录)**: 填写为 `.` (即根目录)。
6. 点击 **“Save and Deploy”**。以后每次您往 GitHub 推送代码，Cloudflare 都会自动为您构建并更新网页！
